01 — Product

Everything a scanner should be

Network, DNS, TLS, and HTTP inspection are table stakes. Here's what's on top.

01

Live security grades

nyx audit scores every run 0–100 and hands you an SSL-Labs-style letter grade, plus a live SVG badge for your README.

02

NyxScript

A real language: its own lexer, parser, AST, interpreter, and a static linter that catches typos before a request leaves your machine.

03

A dashboard, not just a CLI

nyx tui is a full terminal app — live scans, a syntax-highlighted script editor with autocomplete, a plugin browser you can edit in place.

04

Safe by design

TCP-connect scans, standard DNS/TLS/HTTP requests — no packet crafting, no raw sockets, nothing that needs elevated privileges.

05

A REST API

A fourth front-end over the same functions — GET /audit/{domain}, GET /badge/{domain}.svg — no scan logic reimplemented for HTTP. SSRF-guarded by default.

06

Continuous watch

nyx watch re-audits on an interval and stays quiet until something actually changes: a new finding, a resolved one, a grade shift.

02 — Language

A language, not a config file

Variables, loops, string interpolation, and a static linter — all before a single request leaves the machine.

audit.nyx
set targets = ["example.com", "example.org"]

foreach target in targets:
    run audit target as result
    if result:
        save result to "{target}-audit.html"
    end
end

06 — FAQ

Questions worth answering up front

Does anything leave my machine when I run the free tier?

No telemetry, no phone-home. nyx audit talks only to the target you point it at (DNS, TLS, HTTP) — nothing is sent to NYXOR.

Is the grade actually meaningful, or marketing?

It's a deterministic score from finding severity counts — same inputs, same grade, every time. The scoring logic is open source, in core/scoring.py.

Can I use this against infrastructure I don't own?

Only audit domains and hosts you're authorized to test. The REST API refuses to scan internal/private/loopback addresses by default (SSRF guard), but that's not a substitute for authorization.

Is Cloud live today?

Not yet. The waitlist button opens an email, not a checkout — we're honest about that on this page. The GitHub Action already gives you scheduled, hosted-runner audits for free right now.

Why NyxScript instead of just YAML?

YAML can't express a loop or a conditional without turning into a templating nightmare. NyxScript is a small real language — variables, foreach, string interpolation — with a linter that catches mistakes before anything runs.
